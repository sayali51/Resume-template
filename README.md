# 🌐 Personal Portfolio Website – Sayali Kale

A clean, modern, and responsive **personal portfolio website** built using **HTML5, CSS3, and Bootstrap 5**. This project represents a structured digital resume with elegant UI design and well-organized sections.

---

## 📌 Features

- ✨ Minimal and modern UI design  
- 📱 Fully responsive layout using Bootstrap  
- 🎯 Structured sections for better readability  
- 🎨 Custom CSS with reusable variables  
- 🧩 Clean and professional typography  
- 📊 Table-based layout for education and skills  
- 💡 Easy to customize and extend  

---

## 🛠️ Tech Stack

- **HTML5** – Page structure  
- **CSS3** – Styling and layout (`style.css`)  
- **Bootstrap 5** – Responsive design and components  
- **Google Fonts** – Typography
