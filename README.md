# Sayali Kale — Personal Resume Website

A clean, minimal personal resume/portfolio webpage built as a Web Technology assignment.

---

## 📌 Overview

This is a static resume website for **Sayali Kale**, an aspiring web developer and Computer Science student at AISSMS College of Engineering. The page presents her education, skills, and a contact form in a polished, editorial-style layout.

---

## 🗂️ Project Structure

```
├── index.html      # Main HTML page
├── style.css       # Custom styles and design tokens
└── README.md       # Project documentation
```

---

## ✨ Features

- **Responsive design** using Bootstrap 5
- **Sticky navigation bar** with smooth scroll links
- **Sections:** About, Education, Skills, Contact
- **Contact form** with name, email, and message fields
- Custom typography using **DM Serif Display** and **DM Sans** (Google Fonts)
- Minimal, editorial aesthetic with a warm off-white background

---

## 🛠️ Tech Stack

| Technology | Purpose |
|---|---|
| HTML5 | Page structure |
| CSS3 | Custom styling & design tokens |
| Bootstrap 5.3 | Responsive layout & components |
| Google Fonts | Typography (DM Serif Display, DM Sans) |
